# React + Vite

node v18.14.2
react v18.2.0

## Getting Started

cloning project

```bash
git clone https://github.com/raizafr/FrontendDevReactjs-MhdRaizaFarhan.git
cd test-orderfaz-raiza
```

install all dependecies

```bash
npm install
```

running development

```bash
npm run dev
```

Open [http://localhost:5173](http://localhost:5173/) with your browser to see the result.
